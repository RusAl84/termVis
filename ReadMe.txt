Зависимости для питона (3.8.1):
rutermextract
pymorphy2

1)start.py - запуск сервера на localhost:8000

2)Перейти на localhost:8000/main.html
стабильно работает на хроме, у остальных могут быть проблемы с текстом

js скрипты в папке scripts, python в cgi-bin
graph_bulder - строит граф и парсит "тезаурус" (страшный код, осторожно)
csv-analyzer - парсит тезаурус, но не используется
data-checker - выставляет параментры для вершин и узлы для графа
d3-v2 и jquery - либы

extract_terms_to_csv - анализирует текст из корня и сохраняет в out-csv для парса d3
save_text - сохраняет текст из поля в корень (костыль из-за кривого аякса)
